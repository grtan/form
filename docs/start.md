## 属性

|    名称     |    类型    | 必填  | 默认值 |                                      描述                                       |
| :---------: | :--------: | :---: | :----: | :-----------------------------------------------------------------------------: |
|   schema    |  `Object`  |  `Y`  |  `-`   |                           描述表单的类JSON Schema对象                           |
|   submit    | `Function` |  `Y`  |  `-`   | 点击提交按钮时，通过校验后会调用该方法。接受`(data)`参数,`data`表示整个表单的值 |
| label-width |  `Number`  |  `N`  |  `80`  |                               label的宽度，单位px                               |
